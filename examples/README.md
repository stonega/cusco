# Examples

This directory holds example provider configs, prompts, skills, and tool definitions as Cusco grows.

## Skills

- `skills/cusco-mcp-setup/`: a `SKILL.md` that teaches an LLM how to configure MCP servers for Cusco. Cusco ships compact built-in MCP setup guidance that is always available to the model; this longer file is useful as a reference or for installing into another agent by copying the folder to `~/.agents/skills/`.
